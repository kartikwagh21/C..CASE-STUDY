# C..CASE-STUDY

Traffic Signal Simulation System (C++)

This project is a simple Traffic Signal Simulation System developed using C++. It simulates the behavior of real-world traffic lights by automatically switching between signal states (Red, Yellow, and Green) based on time intervals. The program demonstrates basic system simulation concepts and loop-based control logic, providing a simple console-based interface to start and stop the signal simulation.

Features : 

- Simulation of traffic signal states (Red, Yellow, Green)

- Automatic signal switching using time-based logic

- Sequential display of traffic signals

- Menu-driven console interface

Tech Stack :

- C++
- Standard Library (iostream, time/delay functions)
